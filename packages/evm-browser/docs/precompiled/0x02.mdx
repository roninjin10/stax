## Inputs

| Byte range | Name | Description |
|-----------:|-----:|------------:|
| `[0; length[` | data | Data to hash with SHA2-256 |

## Output

| Byte range | Name | Description |
|-----------:|-----:|------------:|
| [0; 31] (32 bytes) | hash | The result hash |

 If not enough gas was given, then there is no return data.
