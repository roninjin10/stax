## Inputs

| Byte range | Name | Description |
|-----------:|-----:|------------:|
| `[0; length[` | data | Data to hash with RIPEMD-160 |

## Output

| Byte range | Name | Description |
|-----------:|-----:|------------:|
| [0; 31] (32 bytes) | hash | The result 20-byte hash right aligned to 32 bytes |

 If not enough gas was given, then there is no return data.

| Input | Output |
|------:|-------:|
