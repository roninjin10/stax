## Stack output

0. `address`: the 20-byte address of the sender of the transaction. It can only be an account without code.

## Example

| * | Input | Output |
|--:|------:|-------:|
| `1` | | `0xbe862ad9abfe6f22bcb087716c7d89a26051f74c` |

[Reproduce in playground](/playground?unit=Wei&codeType=Mnemonic&code='ORIGIN'_).

## Error cases

The state changes done by the current context are [reverted](#FD) in those cases:
- Not enough gas.
- Stack overflow.
