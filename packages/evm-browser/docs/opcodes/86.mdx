*Index 1 is top of the stack. See [PUSH](/#60).*

## Stack input

0. ignored value.
1. ignored value.
2. ignored value.
3. ignored value.
4. ignored value.
5. ignored value.
6. `value`: value to duplicate.

## Stack output

0. `value`: duplicated value.
1. ignored value.
2. ignored value.
3. ignored value.
4. ignored value.
5. ignored value.
6. ignored value.
7. `value`: original value.

| * | Input | Output |
|--:|------:|-------:|
| `1` | `0` | `1` |
| `2` | `0` | `0` |
| `3` | `0` | `0` |
| `4` | `0` | `0` |
| `5` | `0` | `0` |
| `6` | `0` | `0` |
| `7` | `1` | `0` |
| `8` | | `1` |

## Error cases

The state changes done by the current context are [reverted](#FD) in those cases:
- Not enough gas.
- Not enough values on the stack.
- Stack overflow.
