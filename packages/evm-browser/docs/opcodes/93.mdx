*Index 1 is top of the stack. See [PUSH](/#60).*

## Stack input

0. `a`: value to swap.
1. ignored value.
2. ignored value.
3. ignored value.
4. `b`: value to swap.

## Stack output

0. `b`: swapped value.
1. ignored value.
2. ignored value.
3. ignored value.
4. `a`: swapped value.

## Example

| * | Input | Output |
|--:|------:|-------:|
| `1` | `1` | `2` |
| `2` | `0` | `0` |
| `3` | `0` | `0` |
| `4` | `0` | `0` |
| `5` | `2` | `1` |

## Error cases

The state changes done by the current context are [reverted](#FD) in those cases:
- Not enough gas.
- Not enough values on the stack.
